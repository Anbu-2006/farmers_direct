# Farmer's Direct — Website Prototype

A digital farm-to-consumer platform prototype, built for the Jain University TD-PCL 2024–25 project.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Landing page with hero, features, how it works, product preview |
| `about.html` | Project background, team, mentors, university info |
| `farmer.html` | Farmer dashboard prototype (add listings, earnings, AI price tip) |
| `consumer.html` | Consumer marketplace with filter sidebar and product grid |
| `admin-login.html` | Secure login page for platform administrators |
| `admin.html` | Admin dashboard for verification, disputes, and payouts |

## Folder Structure
